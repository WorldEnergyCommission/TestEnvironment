# octopus

This custom docker container is used to sniff device data in the local network from supported devices

## **supported devices**

## --- heat pumps

### Herz (manufacturer)

- commotherm LWi-Split
- commotherm LWi-Mono
- commotherm LW-A

### Viessmann (manufacturer)

- Vitocal 250-A
- Vitocal 252-A

### Vaillant (manufacturer)

- aroTherm plus
- flexoTherm
